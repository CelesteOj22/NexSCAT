Métricas alineadas a ISO/IEC 25010:2023

Las métricas recopiladas se mapean a las características de calidad del estándar, brindando una evaluación estructurada y objetiva.

Mantenibilidad

Complejidad ciclomática, cohesión y acoplamiento entre módulos.

Confiabilidad

Cobertura de código, detección de bugs y vulnerabilidades potenciales.

Adecuación funcional

Tamaño funcional, líneas de código efectivas y densidad de comentarios.

Reusabilidad

Herencia, profundidad de árbol y número de hijos por clase.

Flujo de trabajo

¿Cómo funciona NexSCAT?

En tres pasos simples obtenés un análisis completo de calidad de tu proyecto Java.

PASO 01
Subí tu proyecto

Cargá un archivo ZIP o ingresá la URL de un repositorio GitHub público de Java.

PASO 02
Análisis automático

SonarQube y SourceMeter procesan el código en paralelo y extraen más de 170 métricas de calidad.

PASO 03
Explorá y exportá

Navegá los resultados por proyecto, componente o clase. Exportá en XML, JSON o CSV.
